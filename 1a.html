Lists

Ordered lists

Displays items in a numbered sequence (1, 2, 3 … or i, ii, iii …)

Example: Department list

  1. Information technology
  2. AIML
  3. CSE
  4. EEE
  5. ECE
  6. MECH

Unordered lists

Displays items with bullet points

Example: Courses

  • Database
  • Python
  • Java
  • pandas
  • Mern

Nested List

A list inside another list (can be ordered or unordered).

Example: unordered nested list

  • Subjects
    • Maths
    • English
    • Computer Networks

Ordered list in a unordered list

  1. Programming languages
    • C
    • Java
    • Python

Definition list

Used for terms and their definitions.
dl → definition list container
dt → definition term (word)
dd → definition description (meaning of the word)

Example:

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
